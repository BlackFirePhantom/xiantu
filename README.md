# 暗域传说 - 文字在线RPG

一个轻量级文字在线RPG游戏，适合在 2核1G 服务器上运行。

## 游戏特色

- 9个不同区域探索：新手村、幽暗森林、洞穴深处、龙之峰等
- 20+种怪物战斗，回合制自动战斗
- 装备系统：武器、防具可穿戴/卸下
- 消耗品：药水、永久属性卷轴
- 怪物掉落装备和物品
- 升级系统（最高15级）
- 在线聊天频道
- 排行榜
- 多人在线，实时状态同步

## 快速部署

### 方式一：直接运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
python app.py
```

访问 http://服务器IP:5000

### 方式二：Docker

```bash
docker-compose up -d
```

### 方式三：一键脚本

```bash
chmod +x deploy.sh
sudo ./deploy.sh
```

## 资源占用

- 内存：~50MB
- CPU：几乎无负载（仅在请求时消耗）
- 磁盘：< 10MB（SQLite数据库自动增长）
